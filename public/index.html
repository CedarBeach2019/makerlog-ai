Better than Claude Code because...

repo-first

Your repo IS the agent. Not a cloud session that forgets. Git is your database. Clone it anywhere, your agent comes with it.

BYOK any provider

Not locked to one model. Route tasks to the best model for the job — GPT-4o for planning, Claude for writing, DeepSeek for refactoring.

deploy anywhere

Local machine, Docker container, Cloudflare edge, or fully air-gapped. Your agent runs where you need it.

Every fact, memory, and procedure lives in Git. No cloud dependency for your agent's brain. You own it. Version it. Restore it.
